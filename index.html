Dock + WordPress Blog Creation

This Blog post is on how to use Docker to create WordPress Blog. WordPress is Dynamic Blog using PHP.

Install

Install Docker

Add current user to docker user group, logout and log in the host again.

If ${USER} is sucessfully added, you can run the following command without sudo

Run docker image ls to list the hello-world image that you downloaded to your machine.

List the hello-world container (spawned by the image) which exits after displaying its message. If it is still running, you do not need the –all option:

Define and Build the project

Create and cd into a new folder for the blog at localhost

create docker-compose.yml as in https://docs.docker.com/compose/compose-file/

Run docker-compose up -d from your project directory.

This runs docker-compose up in detached mode, pulls the needed Docker images, and starts the wordpress and database containers, Build

Use docker ps to check if wordpress and mysql is running or not.

Open Brower and enter “http://localhost:8000", WordPress should be running on port 8000 of your Docker Host

Follow the WordPress installation instruciton, the admin dashboard will appear.

how to depoly docker local hosted WordPress blog into Github?

Reference

https://docs.docker.com/compose/wordpress/

On Hexo Blog Setup

This Blog post is on how to create hexo blog and deploy it to Github the first time

Background

I started with Ubuntu 18.04 LTS, which has to be upgraded to Ubuntu 20.04LTS for the newest versions of node.js and npm versions.

Ubuntu 18.04 LTS to Ubuntu 20.04LTS upgrade

Follow the prompt to upgrade. Ubuntu upgrade in recent years become painless, especially for LTS upgrades

Install related software

Install nodejs and npm

To check nodejs and npm versions:

Use npm to install hexo

How to start to create hexo Blog

Create and cd into the Blog folder with intended name

Initialize hexo.

The “Hello World” Blog post will be generated as source/_posts/hello-world.md, you can open the source file

and compare with actual blog post by start hexo

to open http://localhost:4000 in the browser.

To create a new Blog post,

My-First-Blog-Post.md will be created under ource/_posts/.

Edit the md file according to the raw tutorial of hexo “Hello World” post.

How to start to create hexo Blog

In github.com, create a respository named after your github.com username which can be found in upper left of “Settings” of github. You can chang the user name in Account if the username is not taken. If username is abcd, the respository will be “abcd.github.io“

Add github http link to _config.yml file:

Use npm to install hexo git deployer

May see

If this is the first time using git on the Linux localhost, will need to set up git gmail address and name:

Deploy the hexo blog in localhost to github io respository:

hexo blog is deployed in https://abcd.github.io

How to install hexo theme example

Colone a theme from github

edit _config.yml file to change theme to downloaded theme, by change “theme:” value from landscape to yilia

Clean, Generate, and Start to preview new themes at locahost

Deploy to git if theme looks OK.
